## Project Overview

**Next Scene:** is a modern, web application prototype designed for efficient movie discovery and watchlist management. Users can explore trending films, view essential details, and organize their personal list of films to watch.

This project focuses solely on **Frontend UI development** using **HTML** and **Tailwind CSS**.
